# GreenDS-CropPrediction
Deep Learning for Iberian Climate-Resilient Agriculture (ULisboa Master's Project)
---

## Dataset Access & Reference
[cite_start]As outlined in the project guidelines, the dataset used in this predictive pipeline is publicly accessible. [cite_start]The code actively streams multi-variate continuous features reflecting indicators managed by the United Nations Food and Agriculture Organization (FAO) and the World Bank[cite: 86].

* [cite_start]**Direct Dataset URL:** `https://raw.githubusercontent.com/datasets/agriculture-vitals/main/data/crop_yield.csv` 
* [cite_start]**Monitored Variables:** Country, Year, Rainfall (mm), Temperature (°C), Pesticides (tonnes), and Crop Yield (hg/ha)[cite: 86].
